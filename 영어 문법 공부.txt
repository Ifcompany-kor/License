영어의 기본 뼈대 : 주어 , 술어 , 목적어 , 보어(주어를 꾸미기 위해 넣는 말) , 수식어(주어,술어,목적어,보어를 꾸미기 위해 넣는 말)
ex_1) The queen ate lunch : 여왕은 점심을 먹었다. : 주어 + 술어 + 목적어
ex_2) I'm hungry. I'm a hungry woman : 난 배고파. 난 배고픈 여자야 : 주어 + 보어
ex_3) This morning the queen ate breakfast in the bedroom : 오늘 아침 여왕은 침실에서 아침을 먹었다. : 수식어 + 주어 + 술어 + 수식어

기본 뼈대를 만드는 재료 : 8품사
동사 : 움직임을 표현하는 말 ( ex.eat,think,stay )
명사 : 생물/무생물,추상적 개념 ( ex.computer,love,water )
대명사 : 명사 대신 들어가 명사 역할을 하는 말 ( ex.it,that,I,you... )
형용사 : 명사를 앞에서 꾸미거나 뒤에서 설명하는 말 ( ex.big,hungry,happy,cold,blue )
부사 : 동사/형용사/부사를 꾸미는 말 ( ex.then,here,very )
전치사 : 위치나 방향 등을 나타낼 때 쓰는 말 ( ex.at,with,by,from )
접속사 : 단어,구,절끼리 이어 주는 말
감탄사 : 강한 강점을 나타낼 때 쓰는 말 ( ex.oh,wow )
한정사 : 어떤/그/모든 등으로 명사의 범위를 한정해 주는 말 ( ex.a,the,some,any,every )

구와 절은 그 자체가 또 하나의 품사
2개 이상의 단어가 모여 단체로 하나의 품사 역할을 하면 구
명사 역할 하면 명사구, 부사 역할을 하면 부사구 , 형용사 역할을 하면 형용사 구
구와 구가 합쳐서 또 하나의 큰 구를 만들수도 있다.
절은 구와 하는 일은 같은데 그 속에 주어+동사를 갖추고 있다

주절과 종속절
주절 : 혼자만 써도 의미 전달에 문제가 없는 절
종속절 : 혼자서는 의미가 통하지 않고 꼭 주절과 같이 있어야 하는 절

문장의 종류
평서문 : 주어 + 동사
부정문
1. be동사일때 : 주어 + be + not + 보어
2. 일반동사일때 : 주어 + do(es)not + 동사원형
의문문
1. be동사일때 : be + 주어 + 보어
2. 일반동사일때 : Do(es) + 주어 + 동사원형
명령문
1. be동사일때 : Be + 보어 / Don't be + 보어
2. 일반동사일때 : 동사원형 / Don't + 동사원형
감탄문
1. What + a + (형용사) + 명사 / How + 형용사
기원문
May + 주어 + 동사원형

주어가 복수인 경우 명사 뒤에 (e)s를 붙인다.
